# 招聘网页傻瓜式编辑说明

这份说明给没有技术基础的同事使用。只要会打开网页、改文字、下载文件，就能完成日常更新。

## 一、日常只需要改哪个文件

日常不要改代码文件。  
请打开：

```text
editor.html
```

这个页面就是内容编辑器。

## 二、修改内容的 3 个步骤

### 第一步：打开编辑器

1. 打开 `recruitment-site` 文件夹。
2. 双击 `editor.html`。
3. 浏览器会出现“招聘网页内容编辑器”。

### 第二步：只改输入框里的文字

可以修改：

- 营业部名称
- 招募名称
- 首页标题、副标题、介绍
- 总经理金句
- 联系人、电话、邮箱、地址
- 招聘要求
- 福利待遇
- 岗位名称、岗位简介、岗位职责
- 合规提示

不要改文件名，不要打开代码文件。

### 第三步：保存并替换

1. 点页面顶部红色按钮：`1 保存并下载内容文件`。
2. 浏览器会下载一个文件，名字必须是：

```text
site-data.js
```

3. 用新下载的 `site-data.js` 替换网站文件夹里的旧 `site-data.js`。
4. 打开 `index.html` 或正式网址，刷新页面查看效果。

## 三、怎么新增岗位

1. 打开 `editor.html`。
2. 找到“岗位方向”。
3. 点击 `新增岗位`。
4. 填写岗位名称、岗位标签、岗位简介、岗位职责。
5. 点 `1 保存并下载内容文件`。
6. 替换服务器上的 `site-data.js`。

岗位职责一行写一条。

## 四、如果暂时不招聘了怎么办

1. 打开 `editor.html`。
2. 找到“岗位方向”。
3. 点击 `删除最后一个岗位`，直到没有岗位。
4. 点 `1 保存并下载内容文件`。
5. 替换服务器上的 `site-data.js`。

保存后，首页会自动显示“当前暂无在招岗位”，申请页也会停止投递。

## 五、每次上线前要检查什么

发布前请确认：

- 电话、邮箱、地址无误。
- 地址从“河北省”开始写。
- 福利、补贴、休市假等内容已确认。
- “行业独享休市假”已标注“无调休”。
- 没有写“保证收益”“高薪保证”“快速晋升”等承诺性表达。
- 岗位信息已给负责人或合规人员审核。

## 六、最容易出错的地方

**下载后文件名变成了 `site-data (1).js` 怎么办？**  
请把文件名改回 `site-data.js`，否则网站不会读取新内容。

**替换后网页没变化怎么办？**  
先确认替换的是网站文件夹里的 `site-data.js`。然后刷新浏览器，必要时使用强制刷新。

**可以直接改 `index.html` 吗？**  
不建议。日常内容都在 `editor.html` 里改，稳定、简单，也不容易改坏页面。

**正式网站的文件怎么替换？**  
如果网站由技术同事维护，把新下载的 `site-data.js` 发给技术同事，让他替换服务器上的同名文件即可。
